# SingleCycle

### File Structure
```
SingleCycle/
├── rtl/                          # Verilog RTL source files
│   ├── lib/
│   │   └── primitives.v          # half_adder, full_adder, MUX, decoder, DFF
│   ├── arith/
│   │   ├── adder64.v             # 64-bit Kogge-Stone Adder
│   │   ├── barrel_shift.v        # LSL/LSR/ASR/ROR barrel shifter
│   │   ├── mul64.v               # 64×64→128-bit Booth-Wallace multiplier
│   │   └── div64.v               # 64-bit multi-cycle FSM restoring divider
│   ├── alu/
│   │   ├── alu.v                 # 64-bit ALU, NZCV flag generating
│   │   └── alu_control.v         # Opcode → ALU op
│   ├── mem/
│   │   ├── reg_file.v            # 32×64-bit register file
│   │   ├── instruction_mem.v     # Instruction ROM
│   │   └── data_mem.v            # Byte-addressable data RAM
│   ├── core/
│   │   ├── pc.v                  # Program Counter
│   │   ├── sign_extend.v         # Immediate sign/zero extension
│   │   ├── control.v             # Main Control Decoder
│   │   ├── branch_unit.v         # B.cond
│   │   └── vector_stub.v         # Vector/SIMD opcode stub (NOP)
│   ├── pipeline/
│   │   ├── pipeline_regs.v       # IF/ID, ID/EX, EX/MEM, MEM/WB registers
│   │   └── hazard_unit.v         # Stall, forwarding, flush
│   └── top/
│       ├── cpu_top_sc.v          # Single-cycle top module (test aşaması)
│       └── cpu_top.v             # Pipelined top module
├── tb/                           # Testbenches
│   ├── lib/
│   │   └── tb_primitives.v
│   ├── arith/
│   │   ├── tb_adder64.v
│   │   ├── tb_barrel_shift.v
│   │   ├── tb_mul64.v
│   │   └── tb_div64.v
│   ├── alu/
│   │   └── tb_alu.v
│   ├── mem/
│   │   ├── tb_reg_file.v
│   │   └── tb_data_mem.v
│   └── top/
│       ├── tb_cpu_top_sc.v
│       └── tb_cpu_top.v
├── mem/                          # Memory Initialization Files ($readmemh)
│   ├── prog.hex                  # Test program (Generated by Rust Assembler)
│   └── data.hex                  # Initial Data
├── sim/                          # Sim output
│   └── (dump.vcd, loglar)
├── rust/                         # Rust tools
│   ├── Cargo.toml                # Workspace
│   ├── assembler/
│   ├── disasm/
│   └── io_bridge/
├── scripts/
│   ├── sim.sh                    # Simulation Runner
```
